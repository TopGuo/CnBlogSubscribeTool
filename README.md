# CnBlogSubscribeTool
CnBlogSubscribeTool can crawl blog home page data at regular intervals.

## Config

- /Config/Mail.json

````
{
  "Name": "CnBlogSubscribeTool",
  "Address": "",
  "Host": "",
  "Port": 25,
  "Password": "",
  "ReceiveList": [] 
}
````

## Email

9 o'clock every day will send yesterday's blogs
